Technical Notes:
• If "Measured Power" is provided, it is used directly for metabolic calculations.
• Otherwise, Power Output (PO) is estimated from ascent velocity using a physical-mathematical model.
• Note on Metabolic Equivalent: The current VO2 estimation assumes an energy equivalent of 5.189 kcal/L O2, which corresponds to a Respiratory Exchange Ratio (RER) of 1.0. This is valid for high-intensity efforts. However, for submaximal or low-intensity exercise where fat oxidation is higher, this model may slightly underestimate the actual oxygen uptake.
